# FDA Document Intelligence AI (RAG)

An AI-powered FDA 483 document intelligence system built using
Retrieval Augmented Generation (RAG).

This project allows users to query FDA inspection PDFs and receive
accurate, document-grounded answers using OpenAI and Pinecone.
